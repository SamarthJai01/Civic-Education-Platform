# ElectED — Election Process Education Platform

![ElectED Mockup](https://raw.githubusercontent.com/username/elect-ed/main/screenshot.png) *(Note: Add your own screenshot here)*

**ElectED** is a premium, interactive civic education platform designed to help citizens understand the complexities of the democratic election process. From voter registration to the final certification of results, ElectED provides a visual and engaging way to learn how democracy works.

## 🌟 Features

- **Interactive Election Timeline:** A 7-step guide through the entire lifecycle of an election.
- **Electoral Systems Comparison:** Explanations of FPTP, Proportional Representation, Ranked-Choice, and more.
- **Civic Knowledge Quiz:** A custom-built interactive quiz with 8 challenging questions and detailed explanations.
- **Searchable Glossary:** A real-time filtered database of 16+ essential political and electoral terms.
- **Modern Design:** A sophisticated "Dark Mode" aesthetic with noise textures, glassmorphism, and smooth scroll animations.
- **Mobile Responsive:** Fully optimized for all screen sizes.

## 🚀 Technologies Used

- **HTML5:** Semantic structure.
- **Vanilla CSS3:** Custom properties (variables), Flexbox, Grid, and keyframe animations.
- **Vanilla JavaScript:** DOM manipulation, Intersection Observer API, and custom quiz logic.
- **Google Fonts:** Playfair Display, DM Sans, and DM Mono.
